# Plain Markdown Skill

## 任务

阅读并理解文档内容，输出规范化后的版本：

- **内容**：纠正错误（OCR 错误、断行、乱码、格式混乱），保留所有原始信息
- **格式**：去掉一切装饰性标记（加粗、斜体、callout、emoji、多余分割线等），只保留传达结构所必需的格式
- **数学公式**：统一为行内 `$...$`、行间 `$$\n...\n$$`，修正格式错误，简化冗余写法
- **HTML**：转换为等效的 markdown 格式（如 `<table>` 转 markdown 表格、`<br>` 转段落换行等）
- **中英文混排**：中文与英文、数字之间加一个空格
- **不改写**：只动格式，不润色、不增删内容

> **核心约束**：禁止通过编程/脚本方式处理文档。应直接阅读理解全文后，一次性输出规范化版本。对于超长文档，可分节读取、逐节输出，但仍不应依赖脚本做批量替换。

## 输出

- 文件命名：原文件名加 `_plain` 后缀，放在同目录
- 处理完成后，简短说明做了哪些主要改动
